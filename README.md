# 🌐 Personal Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

![Portfolio Preview](https://img.shields.io/badge/Portfolio-Live-brightgreen)

## ✨ Features

- 🎨 **Modern Design** - Clean, dark theme with gradient accents
- 📱 **Fully Responsive** - Looks great on all devices
- ⌨️ **Typing Animation** - Dynamic hero text
- 🎯 **Smooth Scroll** - Smooth navigation between sections
- ✨ **Hover Effects** - Interactive elements throughout
- 📊 **Scroll Animations** - Elements fade in as you scroll
- 🎭 **Morphing Blob** - Animated profile image container

## 🛠️ Tech Stack

- HTML5
- CSS3 (with CSS Grid & Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/Harshavardhan0021/portfolio-website.git
```

2. Navigate to the project:
```bash
cd portfolio-website
```

3. Open `index.html` in your browser or use a live server.

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #00D4AA;    /* Main accent color */
    --secondary-color: #6C63FF;  /* Secondary accent */
    --dark-bg: #0D1117;          /* Background color */
    /* ... */
}
```

### Profile Image
Replace the GitHub avatar URL in `index.html`:
```html
<img src="YOUR_IMAGE_URL" alt="Profile">
```

### Projects
Update the project cards in the Projects section with your actual projects.
